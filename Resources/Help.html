Minecraft Building Guide 3D

Version 0.5.0 (Pre-Alpha) | Developer: Komrad Toast

Introduction

Minecraft Building Guide 3D is a powerful voxel shape generator designed to help you plan and visualize complex Minecraft builds before construction. Generate spheres, domes, arches, bridges, towers, and many other architectural forms with precise block-by-block visualization.

The application provides both 3D viewing for overall visualization and 2D layer-by-layer views for actual in-game construction. Material counts, stack calculations, and dimension information help you prepare the right resources before building.

Quick Start Guide

  1. Select a Shape: Use the dropdown menu in the Shapes tab to choose your desired shape type.
  2. Set Parameters: Adjust the size parameters (radius, height, width, etc.) to your specifications.
  3. Configure Options: Enable Hollow mode if you want a shell, adjust wall thickness as needed.
  4. Generate: Click the green "Generate" button to create your shape.
  5. Explore: Use mouse controls to rotate, pan, and zoom around your creation.
  6. Build: Switch to 2D Layer View and use the layer slider to build layer-by-layer in Minecraft.

Interface Overview

The interface is divided into three main areas:

Control Panel (Left)

Contains three tabs:

  • Shapes: Shape selection, parameters, and generation controls
  • View: Camera controls, layer viewing, and rendering options
  • Build: Manual block placement mode and guide plane controls

3D Viewport (Center)

The main viewing area where your shape is displayed. Supports full 3D navigation in normal mode, or 2D top-down view in layer mode.

Status Bar (Bottom)

Displays current control hints and mode information.

2D Layer View

The 2D Layer View is essential for actual Minecraft construction. It shows a single horizontal slice of your shape from directly above, making it easy to count blocks and replicate the pattern in-game.

Enabling 2D Mode

  1. Go to the View tab
  2. Check the "2D Layer View" checkbox
  3. Use the Layer (Y) slider or number input to select which layer to view

2D Navigation

Action Control
Pan Left Mouse or Right Mouse + Drag
Zoom Scroll Wheel
💡 Building Tip
Start from layer 0 (ground level) and work your way up. Each layer shows exactly which blocks to place at that Y-coordinate in Minecraft.

Building Mode

Building mode allows you to manually add or remove individual blocks from your shape. This is useful for customization or fixing small details.

Modes

  • Add Mode: Left-click on any block face to place a new block adjacent to it
  • Remove Mode: Left-click on any block to delete it
  • Off: Normal viewing mode - clicks don't modify the shape
💡 Tip
Camera controls (rotate, pan, zoom) work normally in all build modes. You don't need to switch modes to navigate the view.

Guide Planes

Guide planes are semi-transparent reference planes with grid lines that help you place blocks in empty space. They're essential when building floating structures or adding details that don't connect to existing blocks.

Available Planes

Plane Color Orientation Position Control
XZ Plane Blue Horizontal Y value
XY Plane Red Vertical (front/back) Z value
YZ Plane Green Vertical (left/right) X value
💡 Quadrant-Based Visibility
Like Blender, planes only appear when you're viewing them from the "correct" side. This prevents visual clutter when planes would overlap. Rotate the camera to see planes from different angles.

Hollow Mode

When enabled, shapes are generated as shells instead of solid blocks. This is useful for buildings you want to enter, or when you need to conserve materials.

Wall Thickness

The Wall parameter controls how thick the shell walls are. A value of 1 creates a single-block thick shell, while higher values create thicker walls.

⚠️ Note
Not all shapes support hollow mode. Spiral staircases and some complex shapes are always generated with their natural structure.

Edge Modes

Edge modes affect how circular shapes handle diagonal connections. In Minecraft, diagonal blocks don't physically connect, which can create visible gaps in curves.

Mode Effect Best For
Normal Standard voxelization Most builds
Heavy Inside Extra blocks fill inner diagonal gaps Solid-looking exteriors
Heavy Outside Extra blocks fill outer diagonal gaps Smooth interior walls

Tips & Tricks

🎯 Building Large Structures
For very large builds, generate the shape and use 2D Layer View. Screenshot or print each layer, then work through them systematically in Minecraft.
🏗️ Hollow Structures
When building hollow structures, generate with Wall thickness of 2-3 for structural stability. Single-block walls can look thin and may have lighting issues.
🌉 Multi-Level Bridges
Start with the default 2 levels and adjust. More levels create more dramatic aqueducts, but require more height clearance.
🔄 Auto-Fit View
Keep "Auto-Fit View" enabled to automatically center and zoom the camera when generating new shapes. Disable it if you want to maintain your current viewpoint.
✏️ Fine-Tuning
Use Build Mode to add or remove individual blocks after generation. This is perfect for adding doors, windows, or fixing small imperfections.
📐 Norman Towers
Place Norman Towers at the corners of your Norman Walls for an authentic castle look. Use the same radius for a consistent appearance.
